Code To Create EC2 Instance

This main.tf file defines an AWS EC2 instance using the aws_instance resource. It sets the AMI, specifies the t2.micro instance type, and assigns the name "amazon_linux" with a tag.

Terraform Init

The terraform init command initializes the project by downloading the necessary AWS provider plugin. It prepares the local environment to interact with your AWS account.

Terraform Validate

The terraform validate command checks the configuration for syntax errors. The "Success!" message confirms the code is correct and ready to be applied.

Terraform Plan

The terraform plan command shows what actions Terraform will perform. The output indicates that it will create one new EC2 instance resource.

Terraform Apply

The terraform apply command executes the plan to build the infrastructure. This begins the process of provisioning the EC2 instance in your AWS account.

EC2 Instance in AWS Console

This AWS console view confirms the EC2 instance was successfully created. It shows the instance named "amazon_linux" is in the "Running" state.

Instance Summary

This detailed view provides key information about the running instance, including its public and private IP addresses, instance ID, and other resource identifiers.

Terraform Destroy Plan

The terraform destroy command creates a plan to remove all managed infrastructure. The output lists the EC2 instance as the resource to be destroyed.

Terraform Destroy Execution

After confirming with "yes", Terraform proceeds to delete the EC2 instance. The "Destroy complete!" message confirms the resource has been successfully removed from AWS.
